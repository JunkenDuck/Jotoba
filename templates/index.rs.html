Jotoba is a multilingual Japanese dictionary.

It is easy to find translations for words or kanji, see
example sentences and the way names can be written.

Here are some examples on how to use this page:

  • Press ' / ' to instantly focus the search bar
  • Quickly change the search type by pressing w / s / n / k
  • You can specify your search by typing #kanji, #adjective, #adverb, #verb or #counter
  • You can find verbs that are conjugated: 食べませんでした
  • You can search multiple kanji at once: 足跡

Jotoba is open source. Check out our Github page if you want to contribute or host Jotoba yourself.
Check out our Trello Board aswell if you are interested in upcoming features and what we are currently working on!
